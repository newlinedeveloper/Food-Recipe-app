# Food-Recipe-app
Food Recipe Preparation and Order App developed by using MERN Stack


## Clone the App

```
  https://github.com/newlinedeveloper/Food-Recipe-app.git
  
```

## To run the Server App

```
  cd server
  npm install
  npm start
```

## To run Client app
```
  cd client
  npm install
  npm start
  
```
